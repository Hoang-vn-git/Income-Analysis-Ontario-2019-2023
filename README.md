# Ontario Income Analysis (2019-2023)

## Project Description
This project offers a comprehensive, Python-based analysis of income trends in Ontario, commencing in 2019. It explores average and median income by year, gender, and age group, evaluates government transfer contributions, and generates Excel dashboards and charts for visualization.

Key features:
- Data cleaning and transformation from CSV, handling missing values and duplicates.
- Pivoting and aggregation to analyze income trends.
- Gender and age group comparison of average and median income.
- Evaluation of government transfers such as COVID-19 benefits, EI, CPP/QPP, Child benefits, OAS/GIS, and social assistance.
- Automated Excel dashboards with pivot tables and charts.
- Python-based workflow to enable reproducible analysis.

## Dataset
The data used in this project is sourced from **Statistics Canada**:

- [Table 11-10-0239-01: Average and median total income, by age group, sex and income source, annual](https://www150.statcan.gc.ca/t1/tbl1/en/tv.action?pid=1110023901)
